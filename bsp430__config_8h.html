Detailed Description

BSP430 configuration directives for generic platform support.

This simplifies management of application-specific bsp430_config.h files. First define anything for your application, then include this file.

This file will first include any platform-specific bsp430_config.h files. Platform-specific configuration headers are included based on the definitions of:

  • BSP430_PLATFORM_EXP430G2
  • BSP430_PLATFORM_EXP430FG4618
  • BSP430_PLATFORM_EXP430F5438
  • BSP430_PLATFORM_EXP430F5529
  • BSP430_PLATFORM_EXP430FR5739
  • BSP430_PLATFORM_EM430
  • BSP430_PLATFORM_RF2500T
  • BSP430_PLATFORM_SURF
  • BSP430_PLATFORM_TRXEB
  • BSP430_PLATFORM_CUSTOM

After platform-specific configuration has been read, this file may provide additional BSP430-generic configuration.

Each inclusion will respect previous configuration values.
